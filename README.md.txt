# CodeAlpha Web Scraping Project

## Task 1: Web Scraping

### Project Description
This project focuses on web scraping quotes and their authors from a public website using Python.
The data is collected using the BeautifulSoup library and saved into a CSV file for further analysis.

### Website Used
https://quotes.toscrape.com

### Tools & Technologies
- Python
- Requests
- BeautifulSoup
- Pandas

### Output File
- quotes_data.csv
